One File. Total Freedom.

Redcap is built on Redbean. It is a "Site-in-a-Can" that runs natively on Linux, Windows, macOS, and BSD. No runtime required. No npm install. No database setup.

# Deploy your entire stack in one go
cp mysite.com user@server:/var/www/
./mysite.com -p 80 -*

Cloud-Native CLI

Running Redcap in the cloud is simple. Since it is a static binary, your Docker container is virtually weightless.

FROM alpine
COPY redcap.com /redcap.com
# Run with writing enabled
CMD ["/redcap.com", "-p", "8080", "-*"]

Get Started

Initialize your first atomic instance and start building in seconds:

# 1. Create admin
./redcap.com -- --init-admin="secure_pass"

# 2. Start serving
./redcap.com -p 8080 -*
